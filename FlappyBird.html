Desarrollo de un videojuego web

Este proyecto es un monográfico realizado junto a una compañera de clase para la asignatura  "Desarrollo de videojuegos"  del grado en Ingeniería Multimedia. Para realizarlo usamos WebGL PlayCanvas, un motor web que permite desarrollar videojuegos usando JavaScript como lenguaje de programación.

Ventajas Desarrollo Web

Al ser un proyecto realizado con JavaScript y HTML, tenemos la ventaja de poder embeber el proyecto en nuestra página web, y de tal forma poder jugarlo desde el navegador sin instalar nada.
